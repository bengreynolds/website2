# Assets

This folder is the content source-of-truth for the portfolio.

Place user-provided content here before wiring it into the SPA:
- `assets/images/`
- `assets/docs/`
- `assets/copy/`
- `assets/data/`

Use this folder for:
- headshots
- project screenshots
- gallery images
- resume files
- testimonials
- award or publication references
- case-study copy
- project link data

Keep the site code pointing at these assets only after the content is ready.
